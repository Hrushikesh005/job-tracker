# Job Application Tracker

A full stack web application to track job applications and interview rounds.

## Features
- Add job applications
- Track interview rounds
- Search and sort companies
- Update interview outcomes
- Delete records

## Tech Stack
- Node.js
- Express.js
- MySQL
- HTML / CSS / JavaScript

## Installation

1. Clone the repository

git clone https://github.com/Hrushikesh005/job-tracker.git

2. Install dependencies

npm install

3. Start server

node server.js

Server runs at:
http://localhost:3000
